# Pin-Grid Generator

A lightweight, responsive web application that dynamically generates a randomized Pinterest-style image grid using vanilla JavaScript and CSS Flexbox.

## 🚀 Features
*   **Dynamic Generation**: Automatically creates a 40-item grid on every reload.
*   **Randomization**: Shuffles a collection of curated image URLs seamlessly.
*   **Responsive Layout**: Uses CSS Flexbox with wrapping for fluid screen adaptation.
*   **Clean UI**: Smooth image clipping using border-radius and object-fit properties.

## 🛠️ Tech Stack
*   HTML5
*   CSS3 (Flexbox)
*   JavaScript (ES6+)

## 💻 How to Run
1. Clone this repository:
   ```bash
   git clone https://github.com
   ```
2. Open `index.html` directly in any modern web browser.
